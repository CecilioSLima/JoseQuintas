# JoseQuintas

Parte do aplicativo Jose Quintas

Na pasta tests vai gerar o aplicativo com tudo que tem nos fontes.
O menu contém opções do que existem nos fontes.

Tem:

- Classe pra PDF
- Preview de relatórios
- Classe pra RMChart
- Classe pra cadastros e outros
- Atualização automática de arquivos/estruturas
- Help on line automático
- Manual em PDF automático
- Browse pra todo uso
- Cadastros de uso geral
- Copiar/colar
- Configurar tamanho de fonte
- Configurar tabela de cores
- Controle bancário pessoal
- Etc etc etc
